# Transformer Architecture Implementation

A complete Transformer implementation from scratch using PyTorch, based on the seminal paper "Attention Is All You Need" (Vaswani et al., 2017).

## 🏗️ Architecture Overview

The Transformer is a revolutionary neural network architecture that relies entirely on self-attention mechanisms, dispensing with recurrence and convolutions entirely.

### Key Components

| Component | Description |
|-----------|-------------|
| **Scaled Dot-Product Attention** | Core attention mechanism: `Attention(Q,K,V) = softmax(QK^T / √d_k) V` |
| **Multi-Head Attention** | Parallel attention heads for different representation subspaces |
| **Positional Encoding** | Sinusoidal position embeddings for sequence order awareness |
| **Feed-Forward Network** | Two-layer MLP applied to each position independently |
| **Encoder Layer** | Self-attention + FFN with residual connections and layer norm |
| **Decoder Layer** | Masked self-attention + cross-attention + FFN |

## 🚀 Quick Start

```python
import torch
from transformer import Transformer, create_causal_mask, create_padding_mask

# Create model
model = Transformer(
    src_vocab_size=10000,
    tgt_vocab_size=10000,
    d_model=512,
    num_heads=8,
    num_encoder_layers=6,
    num_decoder_layers=6
)

# Prepare inputs
src = torch.randint(0, 10000, (2, 10))  # (batch, src_len)
tgt = torch.randint(0, 10000, (2, 8))   # (batch, tgt_len)

# Create masks
src_mask = create_padding_mask(src)
tgt_mask = create_causal_mask(8)

# Forward pass
output = model(src, tgt, src_mask, tgt_mask)
print(output.shape)  # (2, 8, 10000)
```

## 🔧 Architecture Details

### 1. Scaled Dot-Product Attention

The attention mechanism computes a weighted sum of values based on query-key similarity:

```
Attention(Q, K, V) = softmax(QK^T / √d_k) × V
```

The scaling factor `√d_k` prevents dot products from growing too large.

### 2. Multi-Head Attention

Allows the model to attend to information from different representation subspaces simultaneously:

```
MultiHead(Q,K,V) = Concat(head_1, ..., head_h) × W_o
where head_i = Attention(Q×W_q, K×W_k, V×W_v)
```

### 3. Positional Encoding

Since Transformers have no recurrence, positional encodings inject sequence order information:

```
PE(pos, 2i)   = sin(pos / 10000^(2i/d_model))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d_model))
```

### 4. Feed-Forward Network

Applied to each position independently:

```
FFN(x) = ReLU(x × W_1 + b_1) × W_2 + b_2
```

### 5. Encoder Layer

Each encoder layer has two sub-layers:
- Multi-head self-attention
- Position-wise feed-forward network

Each sub-layer uses:
- Residual connection
- Layer normalization

### 6. Decoder Layer

Each decoder layer has three sub-layers:
- Masked multi-head self-attention (causal)
- Multi-head cross-attention over encoder output
- Position-wise feed-forward network
